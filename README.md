# Supply Chain Analytics Dashboard (Power BI)

## Overview
This project presents an interactive Power BI dashboard designed to analyze supply chain performance. It provides insights into demand distribution, supplier efficiency, delivery delays, and product risk levels.

## Key Features
- KPI tracking (Profit, Demand, Delay %, High Risk Products)
- Demand analysis by product category
- Supplier performance based on delay percentage
- High-risk product identification using delivery time
- Interactive filters (Category & Supplier)
- Top N analysis and DAX measures

## Tools Used
- Power BI
- DAX
- Data Visualization

## Key Insights
- Grocery category has highest demand
- Certain suppliers contribute to delays
- High-risk products are linked to longer delivery times
- Opportunity to optimize supplier performance
